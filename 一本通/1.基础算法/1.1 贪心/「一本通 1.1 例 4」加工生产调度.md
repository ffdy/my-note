# 「一本通 1.1 例 4」加工生产调度

`一本通` `贪心`

### 题解

> - 流水作业调度问题
>  	- Johnson算法(~~比较迷,反正我没懂~~)
>  	- 按$min(i.a,j.b)<min(i.b,j.a)$排序
>  	- 枚举记录$ans$

### Coding:
```cpp
#include<cstdio>
#include<cstring>
#include<algorithm>
using namespace std;
const int maxn=1010;
int n;
struct fy{int x,y,w;}q[maxn];
int main()
{
	scanf("%d",&n);
	for(int i=1;i<=n;i++)scanf("%d",&q[i].x);
	for(int i=1;i<=n;i++)scanf("%d",&q[i].y);
	for(int i=1;i<=n;i++)q[i].w=i;
	for(int i=1;i<n;++i)for(int j=i+1;j<=n;++j)
	if(min(q[i].x,q[j].y)>min(q[i].y,q[j].x)) swap(q[i],q[j]);
	int ans=0,bns=0;
	for(int i=1;i<=n;i++)
	{
		ans+=q[i].x;
		bns=max(bns,ans);
		bns+=q[i].y;
	}
	printf("%d\n",bns);
	for(int i=1;i<=n;i++)printf("%d ",q[i].w);
	return 0;
}
```
